# Daily Brief Protocol

Daily briefs are written to:

`content/logs/daily/YYYY-MM-DD-daily-brief.md`

## Required Sections

- Executive Summary
- Revenue Actions
- Content Actions
- Repo Changes
- Airtable State
- Notion State
- Obsidian / Graphify Notes
- Blockers / Needs Major
- Tomorrow's Top 3

For now, the daily brief is generated from local files only.
